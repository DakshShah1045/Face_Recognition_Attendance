How to Use

1 Download the App: Click the download button above to get the .exe file from GitHub Releases.
2 Handle SmartScreen Warning: If Windows shows "Windows protected your PC (SmartScreen)", click "More info" then "Run anyway".
3 Camera Access: The app will automatically open your camera and begin face recognition.
4 Exit the App: Press the Q key on your keyboard anytime to quit the application.
5 Auto-Training: Every time the camera starts, the model automatically trains with the latest data for improved accuracy.
6 Data Export: Attendance data is automatically saved to an Excel file in the app directory.

Frequently Asked Questions

Why is Windows blocking the app?

Windows SmartScreen shows this warning for new applications that don't have an established reputation yet. This is a standard security feature. The app is completely safe - just click "More info" then "Run anyway" to continue.

Do I need to install Python or any dependencies?

No! The .exe file is completely standalone and includes all necessary components. You don't need Python, libraries, or any other software installed.

Where is the attendance data saved?

Attendance records are automatically saved to an Excel file in the same folder as the application. The file includes timestamps and recognized person details.

What cameras are supported?

The app works with any standard webcam or built-in laptop camera. It automatically detects and uses your default camera device.

How accurate is the face recognition?

The system uses advanced AI models with high accuracy rates. The model continuously improves as it processes more data from your specific environment and lighting conditions.
